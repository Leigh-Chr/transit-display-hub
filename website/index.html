Technical details

Architecture

Self-hosted platform. Backend API with WebSocket for real-time updates. PostgreSQL database. Web-based admin panel and display clients.

Deployment

Docker Compose for single-command deployment. All services (API, database, frontend) packaged together.

Stack

  • ws Real-time WebSocket updates
  • pg PostgreSQL persistence
  • docker Containerized deployment
  • rest RESTful admin API
  • svg Network map rendering
